# Contributing Guidelines

## How to Contribute
1. Fork repository
2. Tạo branch mới: `git checkout -b feature/ten-tinh-nang`
3. Commit thay đổi: `git commit -m "Add new feature"`
4. Push lên branch: `git push origin feature/ten-tinh-nang`
5. Mở Pull Request

## Pull Request Process
- Đảm bảo code đã được kiểm thử.
- Cập nhật tài liệu liên quan.
- Tuân thủ coding style của dự án.
- Một Pull Request cần ít nhất **1 review** trước khi merge.

## Development Setup
- Clone repo: `git clone https://github.com/CodelikeC/CS-3332---Finding_apartment_For_students`
- Cài đặt dependencies theo hướng dẫn trong README.
- Chạy test: `npm test` hoặc `pytest` tùy ngôn ngữ.

## Community
Chúng tôi khuyến khích mọi người tham gia thảo luận qua Issues và Pull Requests. Hãy giữ thái độ tích cực và xây dựng.
